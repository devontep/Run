# RunForge

RunForge is a standalone mobile-style running dashboard for an 8-week 2-mile training plan.

## Features

- Built-in 8-week 2-mile improvement plan
- Workout checkoffs
- Manual run logging
- Automatic pace calculation
- Weekly mileage tracking
- Progress charts
- Goal badges
- Local offline-first storage
- Installable from a phone browser when hosted over HTTPS

## GitHub Pages

1. Put these files at the root of a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Choose the default branch and root folder.
5. Open the GitHub Pages URL on your phone.
6. Use **Add to Home Screen** from the browser share menu.

## Local Development

This app is static. You can open `index.html` directly, or serve the folder with any static server.
